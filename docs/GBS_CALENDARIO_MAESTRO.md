# GBS Logistics - calendario maestro

Fuente oficial para revalidar reuniones GBS: `sam@gbs-logistics.cl`.

## Enlaces

- Calendario Google: https://calendar.google.com/calendar/u/0?cid=c2FtQGdicy1sb2dpc3RpY3MuY2w
- Embed público: https://calendar.google.com/calendar/embed?src=sam%40gbs-logistics.cl&ctz=America%2FSantiago
- ICS público: https://calendar.google.com/calendar/ical/sam%40gbs-logistics.cl/public/basic.ics

## Uso operativo

- Todas las reuniones GBS deben existir en el calendario maestro `sam@gbs-logistics.cl`.
- El dashboard debe contrastar GHL/Supabase contra este calendario cuando falten reuniones.
- El ICS privado no se guarda en este documento. Si se necesita como respaldo técnico, usar la variable local `GBS_MASTER_CALENDAR_PRIVATE_ICS`.
